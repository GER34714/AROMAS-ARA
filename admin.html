⚡ Acciones Rápidas

📦 Gestión de Productos

Administrá todo tu catálogo de productos de forma sencilla y organizada

📖

Guía Rápida de Uso

1

Agregar Nuevo Producto

Hacé click en el botón "➕ Nuevo Producto" para abrir el formulario de creación. Completá todos los campos: nombre, categoría, descripción, precio y estado.

2

Buscar y Filtrar

Usá la barra de búsqueda para encontrar productos específicos. Los filtros por categoría y estado te ayudan a segmentar tu catálogo.

3

Editar o Eliminar

Cada producto tiene botones de acción. Editá la información o eliminá productos que ya no necesites.

💡

Consejos Profesionales

✨
Imágenes de calidad: Usá fotos claras y bien iluminadas para mostrar mejor tus productos.
🏷️
Etiquetas personalizadas: Agregá etiquetas como "Más Vendido" o "Nuevo" para destacar productos especiales.
📊
Estados claros: Mantené actualizado el estado de cada producto (Disponible, Por Encargue, Agotado).

🏷️ Gestión de Categorías

Organizá tus productos por categorías para una mejor navegación

📖

Guía de Categorías

1

Crear Categorías

Hacé click en "➕ Nueva Categoría" para agregar nuevas líneas de productos a tu catálogo.

2

Reordenar

Arrastrá y soltá las categorías para cambiar su orden de aparición en la tienda.

3

Editar o Eliminar

Modificá el nombre o eliminá categorías que ya no necesites.

💡

Mejores Prácticas

🎯
Nombres claros: Usá nombres descriptivos que tus clientes entiendan fácilmente.
📸
Imágenes representativas: Cada categoría puede tener su propia imagen para mejor visualización.
🔄
Orden lógico: Organizá las categorías por importancia o tipo de producto.

⭐ Productos Destacados

Gestioná los productos que se mostrarán de forma especial en tu tienda

📖

Cómo Funcionan los Destacados

1

Seleccionar Productos

Elegí hasta 6 productos para destacar. Estos aparecerán en una sección especial en la tienda.

2

Establecer Prioridad

Arrastrá los productos para ordenarlos. El primero será el más destacado.

3

Activar/Desactivar

Usá el interruptor para activar o desactivar el estado destacado de cada producto.

💡

Estrategias de Destacados

🔥
Productos más vendidos: Destacá tus productos con mayor demanda para aumentar ventas.
🆕
Lanzamientos recientes: Promocioná nuevos productos para darles visibilidad.
⭐
Productos premium: Destacá artículos de alta gama o exclusivos.
